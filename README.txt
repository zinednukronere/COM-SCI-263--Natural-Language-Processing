This repository contains the code and the report for the final project of the course COM SCI 263: Natural Language 
Processing. The goal of the project is to perform multi-label tweet emotion identification using the following
ML techniques:

Random Forest with Universal Sentence Encoder
LSTM
Transformer trained from scratch
BERT

The best Jaccard score of 0.5484 is obtained by utilizing the pre-trained BERT approach.


CS263_Final_project.ipynb contains all the code used in the project
CS263_Final_Report.pdf contains the report prepared for the project
